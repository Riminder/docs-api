# Interest JSON

This object represents detected hobbies and interests.

## The Interest Object

```python
{
      'name': 'football', 
      'value': 0.6
}
```

## Attributes

| name | description |
| :--- | :--- |
| name | The interest's name |
| value | The interest's value, it measures the confidence level of expertise |
