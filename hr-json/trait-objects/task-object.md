# Task JSON

Task represents a piece of work done during experience or education for profiles or needed for a job position

## The Task Object

```python
{
    'name':'task',
    'value': null
}
```

## Attributes

| name | description |
| :--- | :--- |
| name | The Task's name |
| value | The task's value |
