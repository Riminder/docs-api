# Interest
This object helps finding interesting and exciting things in the document.
 It lists hobbies and interests detected in the document.

## The Interest Object

```python
{
      'name': 'football', 
      'value': 0.6
}
```

## Attributes

| name | description |
| :--- | :--- |
| name | The interest's name |
| value | The interest's value |
