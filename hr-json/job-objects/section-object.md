# Section object

## Section

All job's section can be structured and presented as a Section object. 
 
## The Section Object

```python
{
    'description': 'Bac+5',
    'name': 'profile',
    'title': 'Searched Profile'
}
```

## Attributes

| name | description |
| :--- | :--- |
| description | The section's description |
| name | The section's name |
| title | The section's title |
