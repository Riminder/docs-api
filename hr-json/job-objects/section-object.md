# Section JSON

## Section

A job's section can be presented as one structured object.

## The Section Object

```python
{
    'description': 'Bac+5',
    'name': 'profile',
    'title': 'Searched Profile'
}
```

## Attributes

| name | description |
| :--- | :--- |
| description | The section's description |
| name | The section's name |
| title | The section's title |
